Favicon test

Testing different favicon formats, sizes, and theme variants.

Icon files
favicon.ico Contains three sizes: 16×16, 32×32, and 192×192. Each frame shows its size as text.
favicon_16.png 16×16 PNG.
favicon_32.png 32×32 PNG. Shows size prefixed with "P".
favicon_64.png 64×64 PNG. Shows size prefixed with "P".
favicon_128.png 128×128 PNG. Shows size prefixed with "P".
favicon.svg SVG favicon showing the text "svg".
favicon_lightdark.svg SVG showing "svg" on a white or dark background depending on prefers-color-scheme.
