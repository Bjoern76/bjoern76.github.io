Favicon setter

Sets the SVG favicon via data URL — either as a native SVG or as a rendered PNG variant.

SVG SVG · auto theme Direct SVG data URL. The browser resolves the theme natively via prefers-color-scheme.
PNG PNG · auto theme Renders the SVG onto a canvas, reads the system theme, and produces a matching PNG.
PNG PNG · force light PNG in light theme — regardless of the current system setting.
PNG PNG · force dark PNG in dark theme — regardless of the current system setting.
