Notes
The browser chooses the icon closest to its required size, scaled by devicePixelRatio. On a 2× display requesting a 32 px icon, it will prefer the 64 px PNG. Each PNG here shows its actual pixel size as text — if the tab icon shows "P32", the browser picked the 32 px file.
