Favicon type fallback

Three <link rel="icon"> in priority order: SVG → PNG → ICO. Tests whether the browser respects type and falls back correctly.

Priority order & format support
Detection method
SVG support is probed by creating a test <img> with an inline SVG data URL. PNG and ICO support are assumed for all modern browsers. The browser should pick the first format it supports — check the network tab to confirm which file was actually requested.
